Acerca de
¿Cómo funciona?
Usa los feeds RSS oficiales de Apple y la iTunes Search API para detectar en tiempo real:

🟢 Gratis hoy — Apps de pago que aparecen en el top-free de Apple. Si estaban en el top-paid y ahora están gratis, el desarrollador las puso en $0.

🟠 Descuento — Apps con precio reducido detectadas cruzando el top-paid con sus precios actuales vs su precio en el top-grossing.

🔵 IAP — Apps gratuitas con compras integradas actualmente en oferta.
¿Por qué algunos datos fallan?
Apple no tiene una API pública de historial de precios. Para ver descuentos reales necesitás subir este archivo a un servidor — desde ahí todas las APIs responden sin restricciones CORS.

Subilo a GitHub Pages: Settings → Pages → main → / (root).
Tu URL: tu-usuario.github.io/appdeals
País del App Store
Por defecto muestra precios en USD (US). Para cambiar el país editá la variable COUNTRY al inicio del JS en el archivo index.html.
Ejemplos: cr Costa Rica · mx México · es España · ar Argentina
